Configuration

1. Rendre les scripts exécutables

Pour exécuter les scripts, rends-les exécutables en exécutant cette commande depuis la racine du projet :

chmod -R +x .

2. Configurer variables.properties

Crée un compte de service via Adobe Developer Console. Assure-toi que ton compte possède les droits suivants :

  • Cloud Manager API
  • User Management API
  • Experience Platform API

Configure le fichier variables.properties avec les informations suivantes :

api_key=
client_secret=
organization_id=
technical_account_id=
metascopes=
